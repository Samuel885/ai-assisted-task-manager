# AI-Assisted Task Management Platform

## Overview
This project is a web-based task management system designed for students. It includes authentication, CRUD task management, and a rule-based workload balancing feature.

## Features Implemented (Current Sprint)
- User Registration
- User Login
- Logout and Session Handling
- MySQL Database Setup (Users and Tasks tables)
- Protected Dashboard
- Improved UI using CSS

## Technologies Used
- PHP
- MySQL
- XAMPP
- HTML & CSS

## Database Setup
1. Import the file `database.sql` into phpMyAdmin.
2. Start Apache and MySQL using XAMPP.
3. Access the system via:
   http://localhost/taskbalance/login.php

## Project Structure
- config.php – database connection
- register.php – user registration
- login.php – user login
- dashboard.php – protected dashboard
- logout.php – session destroy
- style.css – UI styling
- database.sql – database schema
